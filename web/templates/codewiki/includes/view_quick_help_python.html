Return data for a scraper

Get all data
data = scraperwiki.datastore.getData(short_name, limit=-1, offset=0)

Use the limit and offset to control how many records are returned

Get by location
data = scraperwiki.datastore.getDataByLocation(short_name, lat, lng, limit=-1, offset=0)

Items are returned ordered by distance away from the specified latitude and longitude. Use the limit and offset to control how many records are returned. To make use of this, the scraper must have used the latlng variable in datastore.save().

Get by date range
data = scraperwiki.datastore.getDataByDate(short_name, start_date, end_date, limit=-1, offset=0)

Use the limit and offset to control how many records are returned. To make use of this, the scraper must have used the date variable in datastore.save().

Get keys for a scraper
keys = scraperwiki.datastore.getKeys(short_name)
