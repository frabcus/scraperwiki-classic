Scraping a web page

Return the source text of a web page as a string:
html = scraperwiki.scrape(url)

Parsing (simple examples)

Extract data using BeautifulSoup
soup = BeautifulSoup.BeautifulSoup(html)
table = soup.find('TABLE_ID')
for td in table.findAll('td'):
    print td.string

Saving data

Save to the datastore:
scraperwiki.datastore.save(unique_keys=[], data={}, date=None, latlng=[None,None])

Saves a dictionary, data, to the datastore. unique_keys is a list of keys appearing in the dictionary used to determine if if a record should be inserted or updated. date and latlng are optional special fields for indexing each record.

Scraper meta data

Save a persistent variable accessible during another run
scraperwiki.metadata.save(key, value)
Retrieve a persistent variable saved by this scraper
scraperwiki.metadata.get(key, default=None)

Geocoding

Convert a UK postcode to a (lat,lng) pair:
lat_lng = scraperwiki.geo.gb_postcode_to_latlng(postcode)
Convert an OSGB easting, northing to a (lat,lng) pair:
lat_lng = scraperwiki.geo.os_easting_northing_to_latlng(easting, northing)
Extract a UK postcode form a string:
postcode = scraperwiki.geo.extract_gb_postcode(string)

Returns False if no postcode can be found

Recommended libraries

  • BeautifulSoup - for parsing HTML and XML. Full documentation (new window)
  • Mechanize - automates submitting forms. Full documentation (new window)
