ScraperWiki functions

scraperwiki::scrape(url)
Returns the source text of a webpage as a string.

ScraperWiki datastore functions

scraperwiki::save(unique_keys, data={}, date=null, latlng=null)
Saves to the datastore, either adding or over-writing data with the same values for unique_keys. date and latlng are optional special fields for indexing each record.

ScraperWiki geo functions

scraperwiki::gb_postcode_to_latlng(postcode)
returns (lat,lng) array suitable for passing to the scraperwiki::save function.

ScraperWiki metadata functions

scraperwiki::save_metadata(key, value)
saves a persistent variable accessible during another run.
scraperwiki::get_metadata(key, default=null)
retrieves a persistent variable saved by this scraper.
scraperwiki::save_metadata("chart", value)
sets the googlechart image for the scraper.

Recommended libraries

Simple HTML DOM
HTML DOM parser that allows selection of tags using selectors. docs
Add require 'scraperwiki/simple_html_dom.php'; to the top of your scraper in order to use this library.
