External Ruby libraries supported in ScraperWiki

ScraperWiki supports a number of external Ruby libraries that we recommend for screen scraping, data analysis and data visualisation.

If you would like us to add a library that isn't listed here, please get in touch.

  • Nokogiri Parse HTML with a specialist screen scraping library. docs
  • spreadsheet Read and write Excel spreadsheets. docs
  • FasterCSV Faster at reading comma separated value files than Ruby's standard CSV library. docs
  • PDF::Reader Read PDF files. docs
  • Ruby/ZOOM Binding for the Z39.50 Object-Orientation Model, which lets you query various library and bibliography databases. docs
