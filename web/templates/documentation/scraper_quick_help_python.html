How do I get the text of a web page?

import scraperwiki
html = scraperwiki.scrape("http://scraperwiki.com/hello_world.html")
print html

How do I retrieve the bits I care about?

import BeautifulSoup
soup = BeautifulSoup.BeautifulSoup(html)
for td in soup.findAll('td'):
    print td.string

How do I save that data to the datastore?

for td in soup.findAll('td'):
    scraperwiki.datastore.save(unique_keys=['table_cell'], data={'table_cell':td.string})

How do I geocode data?

postcode = scraperwiki.geo.extract_gb_postcode("10 Downing Street, London SW1A 2AA")
lat_lng = scraperwiki.geo.gb_postcode_to_latlng(postcode)
print lat_lng
