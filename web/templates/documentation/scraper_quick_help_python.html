How do I get the text of a web page?

import scraperwiki
html = scraperwiki.scrape("http://scraperwiki.com/hello_world.html")
print html

How do I retrieve the bits I care about?

import lxml.html
root = lxml.html.fromstring(html)
for td in root.cssselect('td'):
    print lxml.html.tostring(td)

How do I geocode data?

postcode = scraperwiki.geo.extract_gb_postcode("10 Downing Street, London SW1A 2AA")
latlng = scraperwiki.geo.gb_postcode_to_latlng(postcode)
print latlng

How do I save my data to the datastore?

for td in root.cssselect('td'):
    data = {'table_cell': td.text} # save data in dictionary
    data["date"] = "2009-03-02"
    data["latlng_lat"], data["latlng_lng"] = latlng
    # Choose unique keyname
    scraperwiki.datastore.save(unique_keys=['table_cell'], data=data)
