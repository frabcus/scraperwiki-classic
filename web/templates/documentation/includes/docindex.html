Technical features

  • Online code editor, write debug and run Python, Ruby or PHP in your browser
  • ScraperWiki code libraries make scraping and parsing simple
  • Nokogiri, lxml, Mechanize — all the libraries you're used to
  • Store data direct to an SQLite datastore
  • API access to data in JSON, CSV
  • Views for writing simple web apps
  • Scheduled to re-run so your data is always up-to-date
  • Email alerts if your scrapers fail, or someone edits them
  • Built in source control, based on Mercurial
