An important part of scraping is turning string data into structured data. Two very common things this happens with are dates/times and locations.

For more details, read the python-dateutil docs, or the .

Parsing basic dates/times

The easiest way is to use a general purpose function that detects many common date formats, and converts them into a Python datetime object.

import dateutil.parser print dateutil.parser.parse('21 June 2010') # 2010-06-21 00:00:00 print dateutil.parser.parse('10-Jul-1899') # 1899-07-10 00:00:00 print dateutil.parser.parse('01/01/01') # 2001-01-01 00:00:00 print dateutil.parser.parse('Tue 27 Sep 2011 00:25:48 BST') # 2011-09-27 00:25:48+01:00 print dateutil.parser.parse('21 June 2010').class # <type 'datetime.datetime'>

This sometimes goes wrong. For example, is this the 2nd March (US) or 3rd February (UK)?

print dateutil.parser.parse('3/2/1999') # 1999-03-02 00:00:00

You can fix it by giving dateutil hints. Or if you really want control, giving a completely explicit format string.

print dateutil.parser.parse('3/2/1999', dayfirst=True) # 1999-02-03 00:00:00 print datetime.datetime.strptime('3/2/1999', '%d/%m/%Y') # 1999-02-03 00:00:00
